# Node-Engineering-Ring-Ding-Dong(NERDD)

노드팀 스터딩딩동 링딩동~~

## 📝Guidance

1. 새로운 스터디 생성시 스터디 디렉토리를 생성해주세요!
2. 생성한 디렉토리 최상단에 `Readme.md` 파일을 생성해주세요. `Readme.md`에는 아래 내용들이 들어가면 좋습니다!
    - 스터디 자료(책, 강의 등)에 대한 ref
    - 스터디 참여 인원
    - 스터디 기간 및 일정
    - 스터디 방식
3. 최상단 `Readme.md`에 디렉토리 ref를 추가해주세요!
